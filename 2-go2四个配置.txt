四个 go2_config 的区别

  文件定位

  ┌────────────────────────────────────────┬────────────────────┬─────────────────────────────────────────────────────┐
  │                  文件                  │        定位        │                      适合场景                       │
  ├────────────────────────────────────────┼────────────────────┼─────────────────────────────────────────────────────┤
  │ go2_config.py                          │ 主力全功能配置     │ 地形课程 + 全课程训练，正式训练用这个               │
  ├────────────────────────────────────────┼────────────────────┼─────────────────────────────────────────────────────┤
  │ go2_config_vanilla.py                  │ 简化基准配置       │ 对标 unitree_rl_gym 原版，关闭大多数高级功能        │
  ├────────────────────────────────────────┼────────────────────┼─────────────────────────────────────────────────────┤
  │ go2_config_vanilla_with_dynamic_cmd.py │ vanilla + 动态命令 │ 比 vanilla 多开了动态命令重采样，但无命令课程       │
  ├────────────────────────────────────────┼────────────────────┼─────────────────────────────────────────────────────┤
  │ go2_config_fast_flat_move.py           │ 高速平地训练       │ 专注平地快速移动，地形设为 plane，速度推进到 4.5m/s │
  └────────────────────────────────────────┴────────────────────┴─────────────────────────────────────────────────────┘

  核心配置差异对比

  1. 训练时长 / Episode

  ┌──────────────────┬────────────┬──────────┬─────────────────┬───────────────┐
  │                  │ go2_config │ vanilla  │ vanilla_dynamic │   fast_flat   │
  ├──────────────────┼────────────┼──────────┼─────────────────┼───────────────┤
  │ episode_length_s │ 25s        │ 20s      │ 25s             │ 25s           │
  ├──────────────────┼────────────┼──────────┼─────────────────┼───────────────┤
  │ 地形类型         │ 课程地形   │ 课程地形 │ 课程地形        │ plane（平地） │
  └──────────────────┴────────────┴──────────┴─────────────────┴───────────────┘

  2. 指令（Commands）设计 — 差异最大

  ┌───────────────────────────┬──────────────────┬─────────────┬─────────────────┬──────────────────────────────┐

  ├───────────────────────────┼──────────────────┼─────────────┼─────────────────┼──────────────────────────────┤
  │ heading_command           │ ✗ 直接角速度     │ ✓ 朝向模式  │ ✗               │ ✗                            │
  ├───────────────────────────┼──────────────────┼─────────────┼─────────────────┼──────────────────────────────┤
  │ resampling_time           │ 5s               │ 10s         │ 5s              │ 5s                           │
  ├───────────────────────────┼──────────────────┼─────────────┼─────────────────┼──────────────────────────────┤
  │ zero_command_curriculum   │ ✓ 0→1500 iter    │ ✗           │ ✗               │ ✓                            │
  ├───────────────────────────┼──────────────────┼─────────────┼─────────────────┼──────────────────────────────┤
  │ limit_vel_prob            │ 0.2              │ 0.0         │ 0.0             │ 0.2                          │
  ├───────────────────────────┼──────────────────┼─────────────┼─────────────────┼──────────────────────────────┤
  │ dynamic_resample_commands │ ✓                │ ✗           │ ✓               │ ✓                            │
  ├───────────────────────────┼──────────────────┼─────────────┼─────────────────┼──────────────────────────────┤

  │ 初始速度范围 lin_vel_x    │ [-0.5, 0.5]      │ [-2.0, 2.0] │ [-2.0, 2.0]     │ [-1.0, 1.0]                  │

  3. 奖励函数 — 只有 fast_flat 有区别

  ┌───────────────────────┬──────────────────────┬────────────────────────────────┐
  │                       │ go2_config / vanilla │           fast_flat            │
  ├───────────────────────┼──────────────────────┼────────────────────────────────┤
  │ tracking_lin_vel      │ 1.0                  │ 2.0（鼓励高速）                │
  ├───────────────────────┼──────────────────────┼────────────────────────────────┤
  │ base_height_target    │ 0.38                 │ 0.33（更低重心，适合快跑）     │
  ├───────────────────────┼──────────────────────┼────────────────────────────────┤
  │ dynamic_sigma         │ ✓                    │ ✓                              │
  ├───────────────────────┼──────────────────────┼────────────────────────────────┤
  │ x_command_hip_regular │ 无                   │ -0.5（有前进指令时约束髋对称） │
  └───────────────────────┴──────────────────────┴────────────────────────────────┘

  4. terrain 设置

  - fast_flat_move 独有：mesh_type = 'plane'，完全在平地上训练，不用地形课程
  - move_down_by_accumulated_xy_command：vanilla 关闭，其余三个都开启

  ---
  一句话总结

  - go2_config = 功能最全的正式训练配置，含全套课程机制
  - vanilla = 对标原版 unitree_rl_gym，朝向模式指令，适合复现对比实验
  - vanilla_with_dynamic_cmd = vanilla 的中间版，加了动态命令但去掉了朝向模式
  - fast_flat_move = 专门训高速平地奔跑，速度目标推到 4.5m/s，奖励也专门调了
